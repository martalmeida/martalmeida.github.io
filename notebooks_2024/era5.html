Download e manipulaçao de dados ERA5¶

https://www.ecmwf.int/en/forecasts/datasets/reanalysis-datasets/era5

  • ERA5 a uma reanlise (quinta geraao) do centro Europeu ECMWF (European Centre for Medium-Range Weather Forecasts), produzida pelo C3S (Copernicus Climate Change Service)
  • ERA5 substitui a reanlise ERA-Interim (que terminou a 31 de agosto de 2019)
  • periodo 1950 at ao presente (1950 a 1978 = back extension dataset)
  • dados horrios e mdias mensais de atmosfera, solo e oceano
  • grelha de 30 km, 137 nveis verticais (superfcie at 80 km)
  • dados disponveis em nvies individuais; nveis de pressao, temperatura potencial e vorticidade; nveis do modelo

Obter os dados¶

https://cds.climate.copernicus.eu/cdsapp#!/search?type=dataset

  • pesquisar era5
  • seleccionar ERA5 hourly data on single levels from 1979 to present

  • login
  • seleccionar Download data

  • seleccionar varivel (ex 2m temperature)
  • seleccionar ano, ms, dia e horas (ex 2021, Dezembro, todos os dias, horas 0, 6, 12 e 18)
  • seleccionar regiao
  • seleccionar formato (ex NetCDF)
  • 1) Submit form ou 2) Show API request

def poly2path(mpoly,trans=False):
    vertices=[]
    codes=[]
    for polygon in mpoly:
        if trans:
            coords=[trans.transform_point(i[0],i[1],ccrs.PlateCarree()) for i in polygon.exterior.coords]
        else:
            coords=list(polygon.exterior.coords)
            
        vertices += coords
        codes += [Path.MOVETO if i == 0 else Path.LINETO
                  for i in range(len(coords))]

        for interior in polygon.interiors:
            if trans:
                coords=[trans.transform_point(i[0],i[1],ccrs.PlateCarree()) for i in polygon.exterior.coords]
            else:
                coords=list(interior.coords)
                
            vertices += coords
            codes += [Path.MOVETO if i == 0 else Path.LINETO
                      for i in range(len(coords))]

    return Path(vertices, codes)
