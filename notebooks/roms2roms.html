creating the child clm file¶

The next example creates a clm file using only the first time step of the parent file.

To create also the bry file use the argument bry=bryname.

Another perhaps important argument is Z, the depths of the intermediate step converting s-levels to z-levels. If not provided, it is used a high resolution default Z between zeta max and bottom, with higher resolution near surface.

In [5]:
roms2roms.roms2clmbry(f,'tmp/child_clm.nc',grd1,sparams,times=[0])
